# Privacy Policy (개인정보처리방침)

## 1. 수집하는 정보
본 서비스는 기본적으로 사용자의 개인 식별 정보를 서버에 저장하지 않습니다. 단, 다음의 경우에 한해 정보를 수집할 수 있습니다:
- API 연결을 위한 브라우저 로컬 저장소(localStorage) 사용
- 서비스 개선을 위한 익명의 사용 통계 (Google Analytics 등 사용 시)

## 2. 정보의 보호
- 사용자가 입력한 API Key 등 민감 정보는 사용자의 브라우저에만 저장되며, 본 서비스의 서버로 전송되지 않습니다.

## 3. 제3자 제공
- 본 서비스는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다.
